# coffee_shop_landingPage
A modern and responsive **Coffee Shop Landing Page** built using HTML5 and CSS3. This project was created to improve my frontend development skills by practicing modern web layouts, Flexbox, CSS Grid, animations, and responsive design principles. And help in my Full Stack Developer Journey.
